# Unleash Java Starter

[![Edit in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/Unleash/unleash-sdk-examples/tree/v2/Java)

Here you can easily get started with [Unleash SDK](https://github.com/Unleash/unleash-client-java) for [Java](https://www.java.com/) and experiment.

```sh
cp -n .env.example .env

dotnet watch run
```

Check out [Program.cs](./Program.cs) for the code.
You can set you instance URL and API client token in `.env`.
Set your flag name on line `:15`.

To add a new dependency, run `dotnet add package <package name>`.
