# Unleash Swift SDK

Codesandbox not supported yet.

Here you can easily get started with [Unleash SDK](https://github.com/Unleash/unleash-ios-sdk) for [Swift](https://www.swift.org/) and experiment.

Check out [Sources/UnleashExample/main.swift](./Sources/UnleashExample/main.swift) for the code. Set your flag name and put your instance URL and client key in `.env`.

## Setup

```sh
cp -n .env.example .env
```

## Run locally

```sh
swift run
```
