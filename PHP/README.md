# PHP Starter

[![Edit in CodeSandbox](https://codesandbox.io/static/img/play-codesandbox.svg)](https://codesandbox.io/s/github/Unleash/unleash-sdk-examples/tree/main/PHP)

Quickly get started with [Unleash SDK](https://github.com/Unleash/unleash-php-sdk) using this starter! 

```sh
composer install
cp -n .env.example .env

php -S 0.0.0.0:8080
```

Or with

```sh
docker compose up
```

Then visit [http://0.0.0.0:8080](http://0.0.0.0:8080)

Check out [index.php](./index.php) for the code. Set your flag name in line `:30`. Put you instance URL and API client token in `.env`.

This starter starts a PHP web server on [localhost:8080](http://localhost:8080).
